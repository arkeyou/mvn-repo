# mvn-repo
Maven repository

```xml
<repositories>
    <repository>
        <id>mvn-repo</id>
        <url>https://github.com/arkeyou/mvn-repo/tree/master</url>
        <releases>
            <enabled>true</enabled>
        </releases>
        <snapshots>
            <enabled>true</enabled>
        </snapshots>
    </repository>
</repositories>
```
